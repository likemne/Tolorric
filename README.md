# Модуль Tolorric | Tolorric Module

[English version below](#english-version)

## 📘 Описание

**Модуль Tolorric** — это стартовый шаблон для создания собственного модуля на платформе «1С-Битрикс: Управление сайтом». Он построен с использованием современного ядра D7 и объектно-реляционного отображения (ORM).

Основная цель — предоставить разработчикам быструю основу для создания своих решений, следуя актуальным стандартам и лучшим практикам разработки под Битрикс.

## ✨ Возможности

*   **Базовая структура**: Включает в себя все необходимые для установки файлы (`install/index.php`, `version.php`) и каркас для дальнейшего развития (`lib/`, `include.php`, `options.php`).
*   **Языковые файлы**: Поддержка русского языка «из коробки».
*   **Готовность к ORM**: Подготовленная структура для создания и управления собственными таблицами в базе данных.
*   **Совместимость**: Следует лучшим практикам для модулей, размещаемых в папке `/local/modules/`.

## 🛠️ Установка

1.  **Скачайте модуль**: Клонируйте репозиторий или скачайте ZIP-архив.
2.  **Скопируйте файлы**: Поместите папку с модулем `tolorric.likemne` в директорию `/local/modules/` вашего сайта. Если папки `local/modules` нет, создайте её.
3.  **Установите в админке**: Перейдите в панели управления в раздел **«Marketplace» → «Установленные решения»**. Найдите модуль **«Мой первый модуль Tolorric»** в списке и нажмите «Установить».

## 📘 Description (English)

**Tolorric Module** is a starter boilerplate for creating custom modules for the «1C-Bitrix: Site Management» platform. It's built using the modern D7 kernel and ORM (Object-Relational Mapping).

The main goal is to provide developers with a quick foundation for building their own solutions, following current standards and best practices for Bitrix development.

## ✨ Features

*   **Basic Structure**: Includes all necessary files for installation (`install/index.php`, `version.php`) and a skeleton for further development (`lib/`, `include.php`, `options.php`).
*   **Language Files**: Russian language support out of the box.
*   **ORM-Ready**: A prepared structure for creating and managing custom database tables.
*   **Compatibility**: Follows best practices for modules located in the `/local/modules/` directory.

## 🛠️ Installation

1.  **Download the Module**: Clone the repository or download a ZIP archive.
2.  **Copy the Files**: Place the module folder `tolorric.likemne` into the `/local/modules/` directory of your website. Create the folder `local/modules` if it doesn't exist.
3.  **Install in Admin Panel**: In the control panel, go to **«Marketplace» → «Installed Solutions»**. Find the **«Мой первый модуль Tolorric»** module in the list and click «Install».
